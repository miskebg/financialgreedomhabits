# Financial Freedom Habits Blog

Custom PHP-based blogging platform with AI-generated articles, 
SEO optimization, comment system, and automated posting via OpenAI API.

## Features

- Automatic AI blog posting (cron + OpenAI)
- SEO optimized article pages
- Comment system with admin moderation
- Responsive design
- Google Analytics + AdSense ready
- Git version control integration

## Tech Stack
- PHP 7+
- MySQL
- HTML/CSS/JS
- cPanel Hosting
- GitHub
